# Hillel Java Pro
## Lesson 8: Exception
Demo project

### Get started

1. Build the project
```sh
    mvn clean package
```
2. Either 
run the jar-file from the target directory
```sh
  java -jar target/javapro.exceptions-{VERSION}.jar
```
or alternatively open the `target` directory in a file explorer 
and double-click on a created jar file
